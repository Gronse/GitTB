Dépot

Gestion de Version

Un gestionnaire de versions (ou VCS, pour version control system) est un logiciel qui permet de conserver un historique des modifications et des versions de tous les fichiers d'un projet, ce qui comprends:
  • les modifications apportées aux fichiers
  • qui a créer ces modifications
  • pourquoi
Le gestionnaire est également capable de fusionner les modifications des personnes qui travaillent simultanément sur un même fichier sans risquer de supprimer les modifications des autres.
Contrôler les differentes versions d'un projet est aussi appelé Versionning.

Git

Qu'est-ce que Git

Git est un gestionnaire de versions (ou VCS, pour version control system).Il est décentraliser, c-a-d qu'il ne repose pas sur un serveur centralisé, mais il utilise un système de connexion pair à pair (P2P).

GitHub

Qu'est-ce que GitHub

GitHub est un service web d'hébergement et de gestion de développement de logiciels, utilisant le logiciel de gestion de versions Git.

Créer un compte GitHub

GitHub est un service en ligne qui permet d'héberger des dépots distants (distant Repository) afin de garder un historique délocalisé. Vous pourrez aussi les rendre publics, et chacun pourra y ajouter ses évolutions. Ou bien les garder privés afin de contrôler qui y a accès.
Créer un compte GitHub
GitLab et Bitbucket sont des alternatives possibles à GitHub.

Branche
GitBash
$ git branch
